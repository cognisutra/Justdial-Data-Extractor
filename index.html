Setup Guide

Install in Chrome
in 60 seconds.

No Chrome Web Store needed — load it directly as an unpacked extension in Developer Mode.

1

Download the Extension

Click the Download button and save justdial-extractor-v2.zip to your computer.

2

Unzip the File

Right-click the ZIP → Extract All (Windows) or double-click (Mac). You'll get a folder called justdial-extractor-v2.

3

Open Chrome Extensions

In Chrome, go to chrome://extensions — or Menu → More Tools → Extensions.

4

Enable Developer Mode

Toggle Developer mode ON using the switch in the top-right corner of the extensions page.

5

Load Unpacked

Click Load unpacked → select the extracted justdial-extractor-v2 folder → click Select Folder.

6

Start Extracting!

Open any JustDial search page, click the extension icon in your toolbar, and hit ⚡ Extract Data.

Page must be loaded — Make sure JustDial search results are fully visible before clicking Extract.

📜

Scroll first — The extractor auto-scrolls, but if results seem incomplete, manually scroll down before extracting.

⚠️

"Developer Mode" warning — Chrome may warn about developer extensions on restart. Dismiss it — this is normal for unpacked extensions.

FAQ

Common questions.

Is this extension free?

Yes, completely free. No subscriptions, no usage limits, no hidden charges. Download it, use it forever.

Does it work on all JustDial pages?

It works on any JustDial search results page — restaurants, hotels, doctors, shops, services — anything that shows a list of businesses.

Why do I need Developer Mode?

Since this isn't on the Chrome Web Store, Chrome requires Developer Mode to load local extensions. It's completely safe and a standard process.

Is my data sent anywhere?

No. Everything happens locally in your browser. Your extracted data is stored in Chrome's local storage and never leaves your device.

What data gets extracted?

Business name, phone number, location/area, and business category — exactly as shown on the JustDial search results page.

The extraction returned 0 results — why?

JustDial occasionally updates their page structure. Try refreshing the JustDial page and extracting again. If the issue persists, contact cognisutra.in for an update.
